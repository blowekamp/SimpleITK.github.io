struct itk::simple::ImageTypeToPixelID< TImageType >

A meta-programming tool to query the PixelID property of an "itk image type" at compile type

This structure contains one property, ImageTypeToPixelID<T>::PixelIDType is the "itk image type" of the pixel ID.
